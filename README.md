# 🌿 Gita Life – Web Application

A modern full-stack web application inspired by the teachings of the **Bhagavad Gita**, designed to manage student registrations, admin workflows, and session coordination seamlessly.

🔗 **Live Demo:** https://gita-life-lake.vercel.app/


📦 **Repository:** https://github.com/rishiwaghdev/gita-life.git

---

## ✨ Overview

**Gita Life** is a structured platform that allows students to register for spiritual learning sessions while providing admins with tools to manage users, schedules, and communication.

The goal of this project is to blend **technology + spirituality**, making it easier to organize and scale Gita-based learning programs.

---

## 🚀 Features

### 👨‍🎓 Student Side

* Easy registration form
* Clean and responsive UI
* Smooth user experience

### 🛠️ Admin Panel

* View and manage student data
* Control session scheduling
* Track registrations

### 🔔 Automation (Planned / Extendable)

* WhatsApp reminders for sessions
* Notification system
* Batch/session management

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/rishiwaghdev/gita-life.git
cd gita-life
```

### 2. Install dependencies

```bash
pnpm install
# or
npm install
```

### 3. Setup environment variables

Create a `.env` file and add:

```
PORT=5000
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
```

### 4. Run the project

```bash
pnpm dev
# or
npm run dev
```
